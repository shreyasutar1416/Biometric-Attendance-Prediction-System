# Biometric Attendance Prediction - Commands

1. Clone the Repository
git clone https://github.com/shreyasutar1416/Biometric-Attendance-Prediction-System.git
cd Biometric-Attendance-Prediction-System

2. Install Dependencies
pip install -r requirements.txt
# Install Streamlit (if not included in requirements.txt)
pip install streamlit

3. Run Data Preprocessing
python data_processor.py

4. Train the Model
python train_model.py

5. Run the Application
python app.py

6. Run Streamlit App
streamlit run app.py
